## 서비스 개요
모무는 신촌 지역에서 활동하는 20, 30대들이 맛집 정보 탐색으로 겪는 어려움에 주목했습니다. 맛집 정보 탐색의 어려움을 해결하며 이용자 취향의 맛집을 찾아갈 수 있도록 모무를 고안했습니다. 먹BTI를 통해 개인 음식 성향 조사를 진행하여 개인 맞춤형으로 맛집 큐레이션이 가능하며 유저 간 맛집 큐레이션을 통한 맛집 선택 과정을 간소화하고자 했습니다. 또한 모무는 선택형 질문지를 통해 간편하게 큐레이션을 요청할 수 있도록 서비스를 제공합니다.
